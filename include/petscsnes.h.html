152: /*E
153:     SNESConvergedReason - reason a SNES method was said to
154:          have converged or diverged

156:    Level: beginner

158:    The two most common reasons for divergence are
159: $   1) an incorrectly coded or computed Jacobian or
160: $   2) failure or lack of convergence in the linear system (in this case we recommend
161: $      testing with -pc_type lu to eliminate the linear solver as the cause of the problem).

163:    Diverged Reasons:
164: .    SNES_DIVERGED_LOCAL_MIN - this can only occur when using the line-search variant of SNES.
165:        The line search wants to minimize Q(alpha) = 1/2 || F(x + alpha s) ||^2_2  this occurs
166:        at Q'(alpha) = s^T F'(x+alpha s)^T F(x+alpha s) = 0. If s is the Newton direction - F'(x)^(-1)F(x) then
167:        you get Q'(alpha) = -F(x)^T F'(x)^(-1)^T F'(x+alpha s)F(x+alpha s); when alpha = 0
168:        Q'(0) = - ||F(x)||^2_2 which is always NEGATIVE if F'(x) is invertible. This means the Newton
169:        direction is a descent direction and the line search should succeed if alpha is small enough.

171:        If F'(x) is NOT invertible AND F'(x)^T F(x) = 0 then Q'(0) = 0 and the Newton direction
172:        is NOT a descent direction so the line search will fail. All one can do at this point
173:        is change the initial guess and try again.

175:        An alternative explanation: Newton's method can be regarded as replacing the function with
176:        its linear approximation and minimizing the 2-norm of that. That is F(x+s) approx F(x) + F'(x)s
177:        so we minimize || F(x) + F'(x) s ||^2_2; do this using Least Squares. If F'(x) is invertible then
178:        s = - F'(x)^(-1)F(x) otherwise F'(x)^T F'(x) s = -F'(x)^T F(x). If F'(x)^T F(x) is NOT zero then there
179:        exists a nontrival (that is F'(x)s != 0) solution to the equation and this direction is
180:        s = - [F'(x)^T F'(x)]^(-1) F'(x)^T F(x) so Q'(0) = - F(x)^T F'(x) [F'(x)^T F'(x)]^(-T) F'(x)^T F(x)
181:        = - (F'(x)^T F(x)) [F'(x)^T F'(x)]^(-T) (F'(x)^T F(x)). Since we are assuming (F'(x)^T F(x)) != 0
182:        and F'(x)^T F'(x) has no negative eigenvalues Q'(0) < 0 so s is a descent direction and the line
183:        search should succeed for small enough alpha.

185:        Note that this RARELY happens in practice. Far more likely the linear system is not being solved
186:        (well enough?) or the Jacobian is wrong.

188:    SNES_DIVERGED_MAX_IT means that the solver reached the maximum number of iterations without satisfying any
189:    convergence criteria. SNES_CONVERGED_ITS means that SNESSkipConverged() was chosen as the convergence test;
190:    thus the usual convergence criteria have not been checked and may or may not be satisfied.

192:    Developer Notes: this must match finclude/petscsnes.h

194:        The string versions of these are in SNESConvergedReason, if you change any value here you must
195:      also adjust that array.

197:    Each reason has its own manual page.

199: .seealso: SNESSolve(), SNESGetConvergedReason(), KSPConvergedReason, SNESSetConvergenceTest()
200: E*/
201: typedef enum {/* converged */
202:               SNES_CONVERGED_FNORM_ABS         =  2, /* ||F|| < atol */
203:               SNES_CONVERGED_FNORM_RELATIVE    =  3, /* ||F|| < rtol*||F_initial|| */
204:               SNES_CONVERGED_SNORM_RELATIVE    =  4, /* Newton computed step size small; || delta x || < stol || x ||*/
205:               SNES_CONVERGED_ITS               =  5, /* maximum iterations reached */
206:               SNES_CONVERGED_TR_DELTA          =  7,
207:               /* diverged */
208:               SNES_DIVERGED_FUNCTION_DOMAIN     = -1, /* the new x location passed the function is not in the domain of F */
209:               SNES_DIVERGED_FUNCTION_COUNT      = -2,
210:               SNES_DIVERGED_LINEAR_SOLVE        = -3, /* the linear solve failed */
211:               SNES_DIVERGED_FNORM_NAN           = -4,
212:               SNES_DIVERGED_MAX_IT              = -5,
213:               SNES_DIVERGED_LINE_SEARCH         = -6, /* the line search failed */
214:               SNES_DIVERGED_INNER               = -7, /* inner solve failed */
215:               SNES_DIVERGED_LOCAL_MIN           = -8, /* || J^T b || is small, implies converged to local minimum of F() */

357: /*MC
358:     SNES_NORM_NONE - Don't compute function and its L2 norm.

360:    Level: advanced

362:     Notes:
363:     This is most useful for stationary solvers with a fixed number of iterations used as smoothers.

368: /*MC
369:     SNES_NORM_FUNCTION - Compute the function and its L2 norm at each iteration.

371:    Level: advanced

373:     Notes:
374:     Most solvers will use this no matter what norm type is passed to them.

376: .seealso: SNESNormType, SNESSetNormType(), SNES_NORM_NONE
377: M*/

379: /*MC
380:     SNES_NORM_INITIAL_ONLY - Compute the function and its L2 at iteration 0, but do not update it.

382:    Level: advanced

384:    Notes:
385:    This method is useful in composed methods, when a true solution might actually be found before SNESSolve() is called.
386:    This option enables the solve to abort on the zeroth iteration if this is the case.

388:    For solvers that require the computation of the L2 norm of the function as part of the method, this merely cancels
389:    the norm computation at the last iteration (if possible).

391: .seealso: SNESNormType, SNESSetNormType(), SNES_NORM_FINAL_ONLY, SNES_NORM_INITIAL_FINAL_ONLY
392: M*/

394: /*MC
395:     SNES_NORM_FINAL_ONLY - Compute the function and its L2 norm on only the final iteration.

397:    Level: advanced

399:    Notes:
400:    For solvers that require the computation of the L2 norm of the function as part of the method, behaves
401:    exactly as SNES_NORM_DEFAULT.  This method is useful when the function is gotten after SNESSolve and
402:    used in subsequent computation for methods that do not need the norm computed during the rest of the
403:    solution procedure.

405: .seealso: SNESNormType, SNESSetNormType(), SNES_NORM_INITIAL_ONLY, SNES_NORM_INITIAL_FINAL_ONLY
406: M*/

408: /*MC
409:     SNES_NORM_INITIAL_FINAL_ONLY - Compute the function and its L2 norm on only the initial and final iterations.

413:    Notes:
414:    This method combines the benefits of SNES_NORM_INITIAL_ONLY and SNES_NORM_FINAL_ONLY.
